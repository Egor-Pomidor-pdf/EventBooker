# EventBooker

Сервис бронирования мероприятий на Go с автоматической отменой просроченных броней.  

Позволяет создавать события, бронировать места, подтверждать участие и автоматически освобождать места, если бронь не подтверждена вовремя.

---

## Возможности

- Создание мероприятий (название, дата, количество мест)
- Бронирование мест пользователями
- Подтверждение (оплата) брони
- Автоматическая отмена неоплаченных броней через заданный интервал
- Разделение ролей:
  - 👤 Пользователь — просмотр, бронирование, подтверждение
  - 🛠 Администратор — создание событий, просмотр всех броней
- Простой Web-интерфейс
- REST API для работы с событиями

---

## HTTP API

- `POST /events` — создать мероприятие  
- `GET /events/{id}` — получить информацию о мероприятии  
- `POST /events/{id}/book` — забронировать место  
- `POST /events/{id}/confirm` — подтвердить бронь  

---

## Фоновая обработка

Используется cron-планировщик для запуска фонового процесса, который:

- Проверяет просроченные бронирования
- Отменяет неподтвержденные брони
- Освобождает места для других пользователей

---

## Web-интерфейс

- `/ui/register` — регистрация
- `/ui/login` — вход
- `/ui/user/events` — список мероприятий (пользователь)
- `/ui/admin/events` — управление мероприятиями (администратор)
